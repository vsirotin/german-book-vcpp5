1. Kleine Schriftarten
----------------------

Der Kurs läuft nur korrekt unter der Einstellung "kleine Schriftarten".
Unter Windows 95 / NT 4.0 können Sie dies unter START / EINSTELLUNGEN / 
SYSTEMSTEUERUNG / ANZEIGE / EINSTELLUNGEN / SCHRIFTGRAD überprüfen.


2. Videos
---------

Um den Kurs benutzen zu können, müssen Sie die Video-Komponenten von Windows 95 / NT
installiert haben. Unter Windows 95 können Sie dies unter START / EINSTELLUNGEN /
SYSTEMSTEUERUNG / SOFTWARE / WINDOWS-SETUP / MULTIMEDIA überprüfen. Das Kästchen vor
VIDEOKOMPRIMIERUNG muß angekreuzt sein.

Während ein Video im Programm abläuft, ist der Maus-Zeiger ein kleiner Pfeil mit
Sanduhr darunter. Wenn das Video zu Ende ist, wird der Maus-Zeiger wieder zum
normalen Pfeil.

Neben dem Knopf "ton" ist eine kleine "Lampe". Leuchtet Sie grün, ist der Ton
angeschaltet. Leuchtet Sie rot, ist der Ton abgeschaltet. Beachten Sie bitte, daß
das Video auch weiterläuft, wenn der Ton aus ist.

3. Kurstext kopieren
--------------------

Wenn Sie Teile des Kurstextes weiterverwenden möchten, können Sie den Kurstext im
Textfenster mit der Maus markieren und mit der Tastenkombination STRG-C in die 
Zwischenablage kopieren. Sie können den Text dann in jeden Editor bzw. in jede 
Textverarbeitung mit der Tastenkombination STRG-V einfügen und ausdrucken.
Beachten Sie bitte hierbei die Lizenzbedingungen am Anfang des Kurses.


4. Aufgaben und Beispiele
-------------------------

Nach den meisten Kapiteln des Kurses folgt ein Übungsteil mit einfachen oder 
komplexen Aufgaben zum Inhalt des jeweiligen Kapitels. Die Aufgaben umfassen 
sowohl Multiple Choice-Fragen als auch Programmieraufgaben und sind selbständig 
zu bearbeiten. Für jede Übung wird eine Musterlösung bereitgestellt, die Sie 
sich durch Anklicken des Buttons 'Lösungshinweis' ansehen können. Falls Sie 
bereits Visual C++ besitzen, können sie die Codeteile in Ihr Projekt integrieren, 
testen und erweitern.

Der vorliegende Visual C++-Kurs hat Workshopcharakter. Am Ende einiger Kapitel 
finden Sie Arbeitsanweisungen, um das Erlernte zu testen und zu vertiefen. 
Diese Arbeitsanweisungen wenden sich an diejenigen Benutzer, die bereits 
Visual C++ besitzen. In der Visual C++-Entwicklungsumgebung können Sie die Schritte 
der Projektentwicklung selbst nachvollziehen. Oder Sie integrieren die fertigen 
Quellcodes in Ihr Projekt. Die Quellcode-Dateien finden Sie auf der CD-ROM im 
Verzeichnis \projekte.

Falls Sie kein Visual C++ besitzen, können Sie sich die Quellcodes in einem Editor 
ansehen und die ebenfalls im Verzeichnis \projekte enthaltenen ausführbaren Dateien 
starten.
